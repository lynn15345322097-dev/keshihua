# 制图说明

目前这张 GIS 叠合图使用的是旧版表：

`GIS图_考古报告位置信息提取表.csv`

这份表共 128 条数据，最后编号到 `GIS-DOC051`。

新版表共 159 条数据，最后编号到 `GIS-DOC082`。两份表的字段结构完全一致，旧表里的 128 条没有被修改或删除，新版只是在后面追加了 31 条新资料。

## 一、现在这张图不用推翻

目前图面已经完成了第一版“古今叠合框架”，包括：

- 现代底图范围；
- 宫城/紫微城范围；
- 大内范围；
- 东西隔城、东西夹城；
- 应天门、玄武门、乾元门、明堂、永泰门步廊；
- 九洲池推定范围；
- 武则天雕像、唐宫御宴等现代可感知点。

这版图部分遗址还停留在“示意性框架”，没有把新增考古资料里的证据差异表现出来。

## 二、新版表主要补了什么

新版表新增编号为 `GIS-DOC052` 到 `GIS-DOC082`，主要补了三类内容：

1. 第三册补充：东隔城、安宁门、九洲池、西夹城、水系、宫城形制总结。
2. 第一册补充：皇城范围、皇城东西墙、东城承福门、含嘉门、承福门街、水系复原总图。
3. 制图解释层：九洲池范围、岛屿、水道、西夹城边界、嘉豫门线索等。

这些新增内容不是都要画进主图。下一版应该按证据等级分层处理。

## 五、图面准确度表达建议

不要让所有线条看起来一样确定。建议按证据等级区分：

| 证据等级 | 图面样式 |
|---|---|
| A | 深色实线或实面，高不透明度 |
| A/B | 实线加轻微透明，标注“待配准” |
| B | 虚线或半透明线面 |
| B/C | 淡色背景层，不进入主要视觉层级 |
| C | 只作说明文字或小图，不进入主地图 |

这样可以让图面既有考古依据，也能让观众看出哪些是发掘确认，哪些是推定或解释。

## 六、下一步制图顺序

建议下一版按这个顺序更新：

1. 保留现有图面框架，不重画全部。
2. 先校准九洲池：加入 `GIS-DOC075`、`GIS-DOC076`、`GIS-DOC077`、`GIS-DOC078`。
3. 再校准西夹城：加入 `GIS-DOC079`、`GIS-DOC080`、`GIS-DOC082`。
4. 最后补东侧关系：加入 `GIS-DOC069`、`GIS-DOC073`。
5. 对 `GIS-DOC071`、`GIS-DOC072`、`GIS-DOC081` 使用虚线或推定点，不要作为确定遗址。

核心原则：新版表不是要求把所有遗址都画上去，而是帮助当前图从“示意叠合”升级为“有证据等级的古今叠合图”。
