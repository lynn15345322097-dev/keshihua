# 隋唐洛阳城中轴线资料初查

更新时间：2026-07-07

## 资料目标

本阶段不再泛泛搜集“隋唐洛阳城”材料，而是围绕一个可视化问题整理资料：

> 隋唐洛阳城中轴线在现代洛阳中如何被保留、偏移、重构、展示和遮蔽？

因此每条资料优先判断四件事：

1. 能否支撑一个中轴线节点或线段。
2. 能否落到现代地图上。
3. 证据等级是否清楚。
4. 能否服务“古今关联度”分析，而不只是历史介绍。

## 证据等级

| 等级 | 类型 | 用途 |
|---|---|---|
| A | 考古发掘报告、发掘简报、文物保护规划、官方遗址资料 | 作为空间位置、遗址范围、节点真实性依据 |
| B | 学术论文、专著、历史地理研究 | 作为中轴线解释、复原判断、空间关系分析依据 |
| C | 古籍、方志、城坊考、类书 | 作为名称、制度、礼仪、空间叙事依据 |
| D | 新闻、公众号、视频、知乎、小红书 | 只作线索和传播参考，不能直接当核心证据 |
| E | 商业 GIS、第三方整理数据 | 可作底图或初始数据，但必须回查来源和精度 |

## 资料检索优先级

### 第一优先：硬证据

1. 《隋唐洛阳城 1959-2001 年考古发掘报告》第二册、第三册。
2. 应天门东阙遗址发掘简报或考古报告。
3. 明堂与天堂遗址发掘简报或考古报告。
4. 定鼎门遗址发掘资料、保护展示资料。
5. 九洲池遗址发掘与遗址公园资料。

### 第二优先：空间解释

1. 隋唐洛阳城中轴线研究。
2. 隋唐东都礼仪空间研究。
3. 隋唐洛阳城城市规划/形制布局研究。
4. 洛阳城水系、洛河、天津桥相关研究。

### 第三优先：古籍与历史地理

1. 《唐两京城坊考》东京/外郭城/通济渠相关部分。
2. 《河南志》唐城阙古迹、京城门坊街隅古迹相关部分。
3. 《旧唐书》《新唐书》《资治通鉴》《唐会要》中应天门、明堂、天堂、东都、神都、天津桥等条目。

### 第四优先：现代城市数据

1. OSM/Geofabrik 河南数据：道路、建筑、绿地、水系、POI。
2. 高德/百度地图：POI、地铁站、景区入口、游客路径。
3. 街景/实地图像：判断可见性、标识系统、游客是否能感知轴线。

## 初步判断

中轴线方向不应做成常规“古今滑块导览”，而应做成“证据驱动的轴线重写分析”：

1. 考古轴：依据发掘报告与复原研究确定的古代中轴线。
2. 现代轴：依据道路、景区、交通和开放空间形成的现代路径。
3. 感知轴：公众实际能看见、能进入、能理解的遗址展示链。

项目的知识增量可以放在三条轴的对齐、偏移、转译与断裂，而不是只展示节点故事。
